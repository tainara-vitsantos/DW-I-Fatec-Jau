Tags Semânticas do HTML5

A tag <header> define o cabeçalho de uma página ou seção.

<header>
  <h1>Cabeçalho da página</h1>
</header>

A tag <footer> define o rodapé de uma página ou seção.

<footer>
  Rodapé com informações
</footer>

A tag <nav> define uma seção de navegação.

<nav>
  <a href="#home">Home</a>
  <a href="#contato">Contato</a>
</nav>

A tag <main> define o conteúdo principal da página.

<main>Conteúdo principal</main>

A tag <section> define uma seção temática de conteúdo.

<section>Seção de artigos</section>

A tag <article> representa um conteúdo independente e reutilizável.

<article>Post de blog</article>

A tag <aside> define conteúdo lateral ou complementar.

<aside>Barra lateral</aside>

Tags de Multimídia (HTML5)

A tag <video> incorpora um vídeo na página.

<video controls width="320">
  <source src="video.mp4" type="video/mp4">
  Seu navegador não suporta vídeo.
</video>

A tag <audio> incorpora um áudio na página.

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  Seu navegador não suporta áudio.
</audio>

Tags de Formulário

A tag <form> cria um formulário de entrada de dados.

<form action="/enviar" method="post">
  <label for="nome">Nome:</label>
  <input type="text" id="nome" name="nome">
  <button type="submit">Enviar</button>
</form>

A tag <input> cria campos de entrada de dados.

<input type="text" name="usuario">

A tag <label> define o rótulo para um campo de formulário.

<label for="email">Email:</label>

A tag <button> cria um botão clicável.

<button type="submit">Enviar</button>
